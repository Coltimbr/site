Кто я такой?

Меня зовут Иван. Я создаю цифровые продукты, которые решают реальные задачи — и при этом не ломаются.

В моём арсенале: разработка сайтов с нуля, работа с CMS, создание приложений для Windows на Python, C++ и C#. Дизайн делаю по необходимости — ровно настолько, чтобы проект выглядел аккуратно. 3D и сложную графику не беру, чтобы не создавать ложных ожиданий.

Главное для меня — быстрый и надёжный результат без лишних сложностей. Клиент получает рабочий инструмент, а не «сырую» красивую картинку.

Среди проектов: образовательный портал OGE365, лендинг для агентства LEIB, голосовой помощник, игра в университетском проекте и другие сайты.
